# **App Name**: DebtWise

## Core Features:

- Loan Registration: Register new loans with all relevant details, including loan type (linked/unlinked, short/long term), interest type (simple/compound, fixed/variable), and custom repayment schedules.
- Payment Tracking: Track monthly payments and update loan statuses automatically. System support RTL and hebrew.
- Automated Notifications: Generate automatic alerts for payment delays and other critical events. System support RTL and hebrew.
- Loan Agreement Generation: Create structured loan agreements with all necessary clauses. The agreements must be written in hebrew RTL
- Automated Interest Calculation: Automatically calculate interest (simple/compound, fixed/variable). System support RTL and hebrew.
- Collateral Management: Manage collateral and attach relevant documents, creating a secured loan process
- Monthly Report Generation: Generate automatic monthly reports on loan status and payment history.

## Style Guidelines:

- Primary color: Dark purple (#4B0082) to convey trust and stability in financial matters.
- Background color: Very light purple (#F0E6EF) for a calm, unobtrusive background.
- Accent color: Teal (#008080) to draw attention to key actions and important data.
- Body font: 'Inter', a grotesque-style sans-serif font for clear, modern readability of text.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy feel for titles and headlines.
- The layout must use a mobile-first approach with full responsiveness and support for all types of screens and devices. All content, including dates, numbers, and currencies, must support the Hebrew language and RTL direction.
- Use financial and organizational themed icons
